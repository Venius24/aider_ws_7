# Belle Croissant Lyonnais Dashboard

## Описание
Это веб-приложение на Django, предназначенное для анализа бизнес-данных пекарни "Belle Croissant Lyonnais". Оно позволяет визуализировать информацию о клиентах, отзывах, активности в социальных сетях, посещаемости сайта и программе лояльности.

## Требования
* Python 3.x
* pip

## Установка и запуск

1. **Клонируйте репозиторий** (или скачайте файлы).

2. **Создайте виртуальное окружение**:
   ```bash
   python -m venv venv
   ```

3. **Активируйте виртуальное окружение**:
   * **Windows:**
     ```bash
     venv\Scripts\activate
     ```
   * **Linux/Mac:**
     ```bash
     source venv/bin/activate
     ```

4. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```

5. **Настройка базы данных**:
   По умолчанию используется SQLite. Если вы планируете использовать SQL Server, отредактируйте `config/settings.py` и убедитесь, что драйверы установлены.

6. **Выполните миграции Django**:
   ```bash
   python config/manage.py migrate
   ```

7. **Инициализация данных**:
   Запустите скрипт для создания таблиц в базе данных и импорта данных из CSV файлов:
   ```bash
   python config/setup_database.py
   ```

8. **Запустите сервер разработки**:
   ```bash
   python config/manage.py runserver
   ```

9. **Доступ к приложению**:
   Откройте браузер и перейдите по адресу: `http://127.0.0.1:8000/`

## Структура проекта
* `config/`: Основная конфигурация Django (настройки, URL, WSGI).
* `customers/`: Модели и логика управления клиентами.
* `feedback/`: Модели и логика обработки отзывов.
* `loyalty/`: Модели и логика программы лояльности.
* `social_media/`: Модели и логика взаимодействия в социальных сетях.
* `website/`: Модели и логика аналитики сайта.
* `setup_database.py`: Скрипт для создания таблиц в SQL Server и импорта CSV.
